Les fleur du Mal

Description

Les fleurs du Mal est un recueil de poemes de Charles Baudelaire publier en 1857 il scandalise alors toute la société francaise. Dans ce recueil Baudelaire nous fais decouvrire un nouveau type de poeme: le Spleen. Ce terme est d'ailleurs maintenant utiliser dans la langue francaise pour designer une profonde mélancolie née du mal de vivre.
